# Weather Dashboard

Weather Dashboard is a simple web application that allows users to search for a city and view real-time weather information.

The application uses the OpenWeather API to fetch live weather data and display key weather details.

## Features

- Search weather by city name
- Real-time temperature display
- Humidity and wind speed information
- Dynamic weather icons based on conditions
- Error message for invalid city names

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
